# Ivan Rybokolnikov — Resume Source

This repository contains the declarative source of my resume.

## Philosophy
I believe that career history should be treated like code:
- **Single Source of Truth**: All data is in `resume.json`.
- **Reproducibility**: PDF is automatically generated via GitHub Actions.
- **Version Control**: Every career update is a commit.

## Tech Stack
- **Source**: JSON Resume Schema
- **CI/CD**: GitHub Actions
- **Environment**: NixOS (for local development)
